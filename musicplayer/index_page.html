üìã Setup Instructions

  1. Prepare your audio file: Place your MP3 file in the root folder and name it exactly "audio.mp3"
  2. Upload to Cloudflare Pages: Upload all files (index.html, host.html, player.html, audio.mp3) to your Cloudflare Pages project
  3. Set up the host device: Open the Host Controller on your main device and set a countdown timer (recommended: 30-60 seconds)
  4. Distribute to player devices: Show the QR code or share the generated URL with all devices that need to play audio
  5. Enable audio on each device: Each player device must tap "Enable Audio" to prepare for synchronized playback
  6. Start the parade: All devices will automatically start playing at the exact same time when the countdown reaches zero!

üìå Important Notes

  • All devices should have a stable internet connection during setup
  • Keep devices connected to power if possible to prevent sleep mode
  • Test with a small group first to ensure synchronization works properly
  • The audio file will be cached on each device for optimal performance
  • Use headphones/earbuds for testing to avoid feedback loops

üîß Technical Details

  • Uses Web Audio API for precise timing control
  • Synchronization accuracy: ~50-100ms across devices
  • Supports all modern browsers (Chrome, Safari, Firefox, Edge)
  • No server required - works entirely on static hosting
  • Automatic volume control and wake lock support
